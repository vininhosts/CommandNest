Desktop AI that can actually act on your computer.

CommandNest is an open-source AI assistant for macOS, Windows, and Linux. Open it with a global shortcut, stream answers through OpenRouter, and use local agent tools for files, code, tests, browsers, email, GitHub, shell commands, and MCP servers.

Why people try it

Global shortcut

Open the assistant instantly from anywhere with Option+Space on macOS, Ctrl+Shift+Space on Windows, or Alt+Space on Linux.

Bring your OpenRouter key

Use OpenRouter streaming chat completions, the free router default, and a searchable model list.

Local agent tools

Let the assistant read, edit, test, organize, browse, email, use GitHub, call MCP servers, and run shell commands with confirmation prompts.
