# Privacy

CommandNest sends prompts, conversation context, selected model IDs, and enabled tool-call payloads to OpenRouter when you ask it to use an AI model.

Your OpenRouter API key is stored in macOS Keychain. It is not committed to the repository and is not saved in `UserDefaults`.

Local Agent Mode can read local files, write files, move items, run shell commands, and open files, apps, or URLs using your current macOS user account. CommandNest does not bypass macOS privacy protections; protected folders may still require Full Disk Access, Accessibility, or Screen Recording permission depending on the action.
